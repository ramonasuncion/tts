# Contributing

## Docstring

Required

1. Parameters

```python
:param s: Input string
```

Optional parameter, if obvious

2. Return values

```python
:return: Tuple of masked string and number of replacements
```

Optional return, if obvious

3. One line summary.

Start with a verb and describe what not how.

Goal: Understand the function does without implementation

```python
"""Mask all but the first and last character of a string"""
```

Required

4. Private helpers

One line is enough

Required

5. Exceptions

Optional, if obvious

> **Note:** Write docstrings before implementing so you aren't going back 4 months later trying to understand what you wrote.
